【整体风格】
- 采用**苹果风半透明蓝色渐变**作为主视觉基调，整体专业大气
- 全局**表格、表单、提示弹窗与确认框的样式必须统一**，保持一致性
- 使用Element Plus组件库
- 主色调为蓝色，白色/半透磨砂卡片背景（若支持，可加入轻微毛玻璃效果）

【页面结构】
1. 最外层用 el-card 包裹（卡片可带半透明蓝白渐变）
2. 顶部是操作栏：左边放“新增”等操作按钮，右边放搜索框
3. 中间是数据表格（带边框、隔行变色）
4. 底部是分页控件

【按钮规范】
- 新增按钮：蓝色，带Plus图标
- 删除按钮：红色，带Delete图标
- 表格内操作：用link样式（文字链接）
- 所有按钮都要有图标

【表格规范】
- 使用 el-table，配置 border 和 stripe
- 操作列固定在右侧（fixed="right"）
- 长文本显示省略号，鼠标悬停显示完整内容
- 表头与行样式需与苹果风半透明蓝色渐变统一

【表单规范】
- 使用 el-form，标签宽度100px
- 必填项前面加红色星号
- 弹窗宽度500-600px
- 弹窗背景、边框、阴影需与整体半透明蓝渐变风格统一

【弹窗与确认框规范】
- 所有弹窗（新增/编辑表单弹窗）和确认框（删除确认）样式必须统一
- 使用 Element Plus 的 Dialog 和 MessageBox，但需通过全局样式或类使其符合苹果风半透明蓝色渐变风格
- 确认框采用危险主题（红色确认）

【交互要求】
- 删除操作要有确认提示
- 操作成功要有消息提示
- 加载数据时显示loading状态
